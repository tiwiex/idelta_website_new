Notes on Password Protection

Yes, the folder can be protected, but not just with HTML alone if you want real security.

The solid options are:

Option 1: Web server auth (best if you control Nginx/Apache) Option 2: Frappe route protection / login-required page Option 3: Temporary front-end password gate (only for light deterrence, not real protection)

If you want real protection, the safest path is to put `/proposal` behind server-side auth or serve it through an authenticated Frappe route. A front-end-only password prompt is easy to bypass.
