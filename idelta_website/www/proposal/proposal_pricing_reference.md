# Proposal Pricing Reference

Combined pricing sheet for AIPMS and FEVA. Intended as an editable working document before prices are finalized in the proposal files.

**Working note:** All amounts below are draft NGN figures extracted from the current AIPMS and FEVA HTML proposals. Older Word documents in the workspace were used only as guides and are not the commercial source for these figures.

**Working FX basis:** ₦1,400 per US$1
